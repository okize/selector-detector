Usage:

  selectorDetector [URL] [OPTIONS]

Description:

  Count the CSS rules, selectors and declarations on a web page.

Options:

  -h, --help           Output usage information
  -V, --version        Output version number
  -p, --pretty         Output a 'pretty' version of results

Examples:

  $ selectorDetector http://www.google.com
  $ selectorDetector http://www.github.com -p
